Assistente virtual acadêmico-administrativo

Tire dúvidas com base em documentos institucionais do seu curso. O sistema combina busca por palavras-chave nos materiais e, se configurado, IA generativa para responder em linguagem natural.

Sou estudante Sou professor Sobre o projeto

Como funciona

  1. Você pergunta em linguagem natural, como numa conversa.
  2. O sistema busca trechos nos documentos ligados ao chatbot do curso.
  3. Você recebe uma resposta objetiva; com API de IA, o texto é gerado a partir desses trechos.

Primeiro acesso

No Admin do Django, cadastre cursos, vínculos, chatbots e materiais (com texto pesquisável). Depois entre como estudante com RA e senha para abrir o assistente.

Dica: quanto mais completo o campo texto para busca no material, melhores serão os resultados.
