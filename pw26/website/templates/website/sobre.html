Projeto de pesquisa

Chatbot para atendimento acadêmico-administrativo baseado em documentos institucionais — protótipo para o IFPR Campus Paranavaí.

Muitas dúvidas dos estudantes estão em calendários, regulamentos e normas. Fora do horário da secretaria, esse acesso nem sempre é rápido. O trabalho propõe um assistente virtual que recupera trechos nesses documentos e pode usar IA generativa via API para responder com clareza.

A pesquisa segue Design Science Research (DSR): problema → artefato (este site) → demonstração e avaliação com usuários.

O que este site faz

Implementa o modelo de dados (cursos, pessoas, materiais, chatbots), login de estudante e professor, cadastro de materiais com texto pesquisável e uma tela de conversa alinhada ao fluxo entrada → processamento → saída da metodologia.
